# Bibliotecas instaladas

* Google Fonts
> flutter pub add google_fonts
### Referencias:
https://pub.dev/packages/google_fonts


* Gerenciador de id (uuid)
> flutter pub add uuid
### Referencias:
UUID. https://pub.dev/packages/uuid

# Gerenciamento de estado
